# Originaria WordPress Theme

## Descripción

Tema personalizado para Originaria - Agencia Creativa. Este theme está diseñado como una landing page moderna y elegante basada en el diseño HTML original.

## Características

- **Landing Page Estática**: Diseño de una página con secciones de navegación suave
- **Revolution Slider**: Slider full-screen con efectos de blur
- **Responsive**: Completamente adaptable a dispositivos móviles y tablets
- **Secciones Principales**:
  - Home (Slider)
  - Nosotros
  - Servicios
  - Proyectos
  - Contacto

## Requisitos

- WordPress 5.0 o superior
- PHP 7.4 o superior

## Instalación

1. Sube la carpeta del tema al directorio `/wp-content/themes/`
2. Activa el tema desde el panel de WordPress en Apariencia → Temas
3. El tema está listo para usar

## Estructura de Archivos

```
originaria/
├── css/              # Archivos de estilos
├── js/               # Scripts JavaScript
├── images/           # Imágenes y gráficos
├── fonts/            # Fuentes tipográficas
├── revolution/       # Revolution Slider assets
├── style.css         # Hoja de estilos principal y metadatos del tema
├── functions.php     # Funciones del tema
├── header.php        # Cabecera del sitio
├── footer.php        # Pie de página
├── index.php         # Template principal
└── README.md         # Este archivo
```

## Personalización

Para personalizar el contenido:

1. **Textos**: Edita el archivo `index.php` y modifica los textos directamente
2. **Imágenes**: Reemplaza las imágenes en la carpeta `images/`
3. **Estilos**: Modifica los archivos CSS en la carpeta `css/`
4. **Información de Contacto**: Edita la sección de contacto en `index.php`

## Secciones del Tema

### Home
Slider full-screen con Revolution Slider mostrando 3 slides con imágenes y texto.

### Nosotros
Sección con imágenes y texto descriptivo sobre la agencia.

### Servicios
Grid de 6 servicios principales:
- Marketing
- Digital Ads
- Branding
- Social Media
- Web Development
- Audiovisual

### Proyectos
Portfolio masonry con 10 proyectos de ejemplo.

### Contacto
Formulario de contacto con información de dirección, teléfono y redes sociales.
